## Features

- Adds labels automatically

## How it Works

1. User opens an issue with crypto-related keywords.
2. The bot checks if it has already replied to avoid duplicates.
3. If not, it posts a professional response with guidance and support links.
4. Labels are added automatically based on keywords.
